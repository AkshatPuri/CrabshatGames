Dots...

About the game!

Dots... is a game full of Dots... and different unlockable cannons, Dots is a hyper casual relaxing game with most minimalistic design.

Try every level as every level is randomly generated and unique, patience is require to play through all the levels, Try unlocking different cannons to unlock different color Dots in order to make game more fun.

Features:

  • -600 Randomly Generated Levels.
  • -most Minimalistic Design.
  • -Different unlockable cannons and Dots.
  • -Simple Dotsfull gameplay, relaxing music.
  • -Minimalistic Animations.
  • -Contains well Optimized Ads for better gameplay.
  • -Leaderboard

Category: Casual
